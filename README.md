**Открытая система управления производством и роем роботов.**

>Этот проект — инструмент для тех, кто создаёт, а не для тех, кто покупает. 
>Он создан, чтобы служить тому, кто его собрал и настроил, а не тому, кто платит. Это не просто софт. 
>Это твой цифровой соратник, твой «Урфин Джус», который поможет поддержать твоё дело дело.
>проект сделан на Ubuntu 22.04

**Специализация:** Робототехника и широкая промышленная автоматизация.
**Основы:** Формальная логика, диалектический материализм, инженерный расчёт.

---

### 🚀 Что он умеет уже сейчас

- **Управлять производством:** 3D-принтеры, эмуляция ЧПУ, генерация G-кода.
- **Проектировать:** ядро PicoGK компании Leap 71 для создания 3D-моделей и деталей.
- **Управлять роботами:** ROS2 для координации роботов, дронов и ИИ агентов.
- **Думать и спорить:** Двухполушарный ИИ (Исполнитель-Критик) для поиска ошибок и стратегического планирования.
- **Тактически планировать:** GOAP-мозжечок для быстрых, автономных действий.
- **Долговременная память:** Помнит контекст, учится на ошибках, так же можно отчищать память

### 🛠️ Быстрый старт (на твоём ПК) 

1.  **Клонируй репозиторий:**
    ```bash
    git clone https://github.com/klhhhhh568/uorfin-jus.git
    cd uorfin-jus
    ```
2.  **Создай виртуальное окружение:**
    ```bash
    python3 -m venv uorfin_env
    source ~/uorfin_env/bin/activate  # Linux/Mac
    # или uorfin_env\Scripts\activate для Windows
    ```
3.  **Установи зависимости:**
    ```bash
    pip install -r requirements.txt
    ```
4.  **Настрой ключи:**
    Скопируй файл `.env.example` в `.env` и вставь свой API-ключ OpenRouter.
5.  **Запусти систему:**
    Скомпилируй ядро (C++):
    ```bash
    cd core && mkdir build && cd build && cmake .. && make
    ```
    Запусти веб-интерфейс:
    ```bash
    python web/web_server.py
    ```
6.  **Открой штаб в браузере:**
    Перейди по адресу `http://localhost:5000`. Готово. Ты в веб-интерфейсе. Начинай отдавать приказы.

7.  Так же для быстрого запуска был разработан файл run.sh
    cd ~/uorfin_jus
    ./run.sh
    переёди по ссылке `http://localhost:5000`

### ⚙️ Как добавить новый навык

«Урфин» — модульная система. Ты можешь научить его чему угодно.

1.  Создай Python-файл в папке `skills/` с методами `init(name)`, `handle_message(message)` и `shutdown()`.
2.  Зарегистрируй его в `uorfin_charter.yaml` (секция `tools`).
3.  Перезапусти ядро. Навык готов.
